# Test Repository

This is a simple test repository containing JavaScript utility functions.

## Available Functions

The `sample.js` file provides the following utility functions:

- `add(a, b)` - Adds two numbers together
- `multiply(a, b)` - Multiplies two numbers
- `subtract(a, b)` - Subtracts the second number from the first
- `divide(a, b)` - Divides the first number by the second
- `power(base, exponent)` - Calculates the power of a number

## Usage

```javascript
const utils = require('./sample.js');

// Examples
console.log(utils.add(2, 3));        // 5
console.log(utils.multiply(2, 3));   // 6
console.log(utils.subtract(5, 2));   // 3
console.log(utils.divide(6, 2));     // 3
console.log(utils.power(2, 3));      // 8
```
